# signum
> Fonctions

Détermine le signe du nombre **number**.

#### Paramètres

- **number** Le nombre dont le signe sera déterminé.

#### Retour

- **sign** 1 si le nombre est positif, 0 si le nombre est nul et -1 si le nombre est négatif.

#### Exemples

`signum(0) // 0`
`signum(50) // 1`
`signum(-12) // -1`
