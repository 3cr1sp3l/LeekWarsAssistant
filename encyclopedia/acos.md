# acos
> Fonctions

Calcule l'arc cosinus d'<b>argument</b>, dans l'intervalle [0, [[PI]]].

#### Paramètres

- **argument** Nombre dont l'arc cosinus sera calculé.

#### Retour

- **result** L'arc cosinus d'<b>argument</b>.

#### Exemples

`acos(0) // PI / 2`
`acos(0.5) // PI / 3`
`acos(-0.5) // 2 * PI / 3`
