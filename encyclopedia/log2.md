# log2
> Fonctions

Calcule le logarithme en base 2 du nombre **number**.

#### Paramètres

- **number** Un nombre compris dans l'intervalle ]0; +∞[.

#### Retour

- **log2**
  - Si **number** > 0, le logarithme en base 2 de **number**.
  - Si **number** <= 0, renvoie `NaN`.

#### Exemples

`log2(1) // 0.0`
`log2(10) // 3.3219280948873626`
`log2(16) // 4.0`
