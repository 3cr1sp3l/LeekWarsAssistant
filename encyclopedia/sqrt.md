# sqrt
> Fonctions

Calcule la racine carrée du nombre **number**.

#### Paramètres

- **number** Nombre dont la racine sera calculée.

#### Retour

- **root** Racine carrée de number.

#### Exemples

`sqrt(1) // 1`
`sqrt(2) // 1.41421356237`
`sqrt(4) // 2`
`sqrt(31329) // 177`
