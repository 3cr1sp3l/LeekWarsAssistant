# atan
> Fonctions

Calcule l'arc tangente d'argument, dans l'intervalle [0, [[PI]]].

#### Paramètres

- **argument** Nombre dont l'arc tangente sera calculée.

#### Retour

**result** L'arc tangente d'argument.

#### Exemples

`atan(0) // 0`
`atan(sqrt(3) / 3) // PI / 6`
`atan(1) // PI / 4`
`atan(-1) // -PI / 4`
