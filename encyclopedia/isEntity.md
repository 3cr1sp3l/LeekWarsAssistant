# isEntity
> Fonctions

Détermine si le contenu de la cellule **cell** est une entité.

*Exemple* :
```
if (isEntity(cellule)) {
	// cette cellule contient une entité
}
```

#### Paramètres

- **cell** : La cellule à tester.

#### Retour

- **isEntity** : `true` si la cellule contient une entité, `false` sinon.
