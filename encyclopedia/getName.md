# getName
> Fonctions

Renvoie le nom de l'entité d'id <b>entity</b>.

*Exemple* : saluer l'adversaire :
```
var enemy = getNearestEnemy()
say("Salut " + getName(enemy) + " !")
```

#### Paramètres

- **entity** : L'id de l'entité dont le nom sera renvoyé.

#### Retour

- **name** : Le nom de l'entité <b>entity</b>.
