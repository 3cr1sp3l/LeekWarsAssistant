# randInt
> Fonctions

Renvoie un nombre aléatoire entier, compris entre **a** (inclus) et **b** (exclu).

#### Paramètres

- **a** Borne inférieure.
- **b** Borne supérieure.

#### Retour

- **number** Un nombre entier aléatoire dans l'intervalle [**a**; **b**[.

#### Exemples

`randInt(10, 20) // 14`
`randInt(1000, 2000) // 1680`
