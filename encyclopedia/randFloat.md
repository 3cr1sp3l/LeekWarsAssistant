# randFloat
> Fonctions

Renvoie un nombre aléatoire réel, compris entre **a** (inclus) et **b** (exclu).

#### Paramètres

- **a** Borne inférieure.
- **b** Borne supérieure.

#### Retour

- **number** Un nombre aléatoire dans l'intervalle [a; b[.

#### Exemples

`randFloat(10, 20) // 17.551965161250713`
`randFloat(1000, 2000) // 1610.705920097898`
