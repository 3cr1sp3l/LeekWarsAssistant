# toDegrees
> Fonctions

Convertit l'angle radians en degrés.

#### Paramètres

- **radians** Valeur en radians

#### Retour

- **degrees** Angle en degrés.

#### Exemples

`toDegrees(0) // 0`
`toDegrees(PI / 4) // 45`
`toDegrees(PI / 3) // 60`
`toDegrees(PI / 2) // 90`
`toDegrees(PI) // 180`
`toDegrees(2 * PI) // 360`
