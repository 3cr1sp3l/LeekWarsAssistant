# cos
> Fonctions

Calcule le cosinus de **angle**.

#### Paramètres

- **angle** Nombre dont le cosinus sera calculé (en radians).

#### Retour

- **cosine** Le cosinus de angle, dans l'intervalle [-1, 1].

#### Exemples

`cos(0) // 1`
`cos(PI / 2) // 0`
`cos(PI / 3) // 0.5`
`cos(PI / 6) // sqrt(3) / 2`
