# sin
> Fonctions

Calcule le sinus de l'angle **angle** passé en paramètre.

#### Paramètres

- **angle** L'angle dont le sinus sera calculé.

#### Retour

- **sine** Retourne le sinus de l'angle angle.

#### Exemples

`sin(0) // 0`
`sin(PI / 2) // 1`
`sin(PI / 3) // sqrt(3) / 2`
`sin(PI / 6) // 0.5`
